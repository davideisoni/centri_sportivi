Per far funzionare l'applicazione web è necessario utilizzare xamp, inserendo la cartella centri_sportivi
all'interno di htdocs.
Una volta aperto il localhost si avvierà l'index.php, il quale rimanderà alla pagina di login.
Successivamente al login o alla registrazione si verrà reindirizzati alla applicazione web.
le password degli utenti inseriti nel dump sono criptate, la password per effettuare il login è l'username stesso.
es. username: davide ---> password: davide
All'interno della cartella openme è presente il file da importare su phpmyadmin per avere il database già pronto.
